## Project Summary
BetterPic B2B is a platform designed to provide studio-quality AI headshots for organizations at scale. It targets industries like Healthcare, Legal, and Real Estate, offering a cost-effective and time-efficient alternative to traditional corporate photography.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Shadcn UI
- **Database/Auth**: Supabase
- **CRM/Backend**: Attio
- **Runtime**: Bun

## Architecture
- `src/app`: Contains the main application routes and API endpoints.
- `src/components`: Reusable UI components built with Radix UI and Tailwind.
- `src/lib`: Utility functions and library configurations (Supabase, Attio).
- `src/hooks`: Custom React hooks for state and data management.
